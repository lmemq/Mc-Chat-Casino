# Mc-Chat-Casino
Just a simple chat casino for minecraft servers
(use on ur own risk! this may cause ban / mute on some severs)
(it can be used only for minecraft)

# Installing
- Install forge / fabric / their forks and download `minescript` mod to ./mods folder (also, configure it following minescript wiki) 
- Clone this repo: `git clone https://github.com/lmemq/Mc-Chat-Casino.git` anywhere and put kaz.py to ./minescript
- In the mc, use `\kaz` to start it and `\killjob 1` to stop. or just leave the sever / game
